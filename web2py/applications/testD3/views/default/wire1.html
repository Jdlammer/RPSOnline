Example of drawing a progressive line over a path, but the original path was changed from geojason to a simple svg shape. No D3 used, but example used in later examples with D3.
The svg-editor can make wireframes for backgrounds, and you can include these as backgrounds to any svg, by including the svg definition inside your html page.
Change the attribute "opacity" to a float number from 0 to 1 to make a background transparent.
